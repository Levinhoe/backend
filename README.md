# 慧医数字医疗应用系统

基于 Spring Boot + Vue 3 的前后端分离数字医疗管理系统。

## 功能模块

- **数据面板**：统计卡片（可点击跳转）、职称/科室图表、最新政策、库存预警、系统公告
- **用户管理**：增删改查、重置密码、修改密码、Token 鉴权
- **医生信息管理**：增删改查、科室筛选、下拉选择科室、重置密码
- **医药公司管理**、**医药公司政策管理**（公司筛选、关联下拉）
- **城市信息管理**、**销售地点管理**（城市筛选、地图展示、表单校验）
- **药品信息管理**（分类筛选、低库存筛选、库存标签预警）
- **必备材料管理**、**企业信息管理**、**合作伙伴管理**
- **公告内容管理**（状态筛选、一键发布/撤回）

## 快速启动

```bash
# 1. 导入数据库
mysql -u root -p123456 < database.sql

# 2. 启动后端
cd backend && mvn spring-boot:run

# 3. 启动前端
cd frontend && npm install && npm run dev
```

访问 http://localhost:5173 ，账号 `admin` / `123456`

## 技术特性

- 登录 Token 鉴权，接口自动携带 Authorization
- 统一 CRUD 组件（表单校验、标签列、密码字段、刷新、空状态）
- 青绿主题 `#009688`，Element Plus 主色统一
- 后端 REST API 完整，支持多条件筛选
